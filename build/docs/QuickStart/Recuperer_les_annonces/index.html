Récupérer les annonces

Important

L'URL racine est ici https://carrers.flatchr.io

Requête​

Il est possible de récupérer vos annonces au format JSON en utilisant la requête suivante : 

GET /company/{slug}.json

Paramètres​

ParamètreInTypeObligatoireDescription
slugpathstring
✔️
Slug de l'entreprise

Exemple de requête​

Requête cURL pour Flatchr
curl -X GET https://careers.flatchr.io/company/flatchr.json
    -H "Authorization: Bearer {token}"

Réponse​

NameTypeDescription
idstringClé de la diffusion
offer_idintegerIdentifiant offre jobboards
external_id
publishedbooleanL'annonce est elle publiée ou non
created_attimestampDate de création de la diffusion
vacancyvacancy
vacancy_idstringIdentifiant de l'annonce
